## 💡 開發提示

### 技術複用策略
1. **隨機工具**: 擲硬幣、擲骰子、輪盤 → 共用隨機引擎
2. **資料產生器**: 手機號、電話號、身份證 → 共用驗證邏輯
3. **文字工具**: 刪除線、Instagram 字體 → 共用 Unicode 處理
4. **測試工具**: 壞點測試、色弱測試 → 共用全螢幕框架

### 優化建議
- 所有工具都支援一鍵複製到剪貼簿
- 娛樂工具加入分享到社群功能
- 資料產生器標註「僅供測試使用」
- 建立工具收藏功能 (LocalStorage)

### i18n 準備
每個工具至少準備:
- 繁體中文
- 英文
- (可選) 簡體中文
